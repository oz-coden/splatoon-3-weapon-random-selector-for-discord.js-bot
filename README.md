## Summary

This contains Japanese Splatoon Weapon Name!  
So if you want to use in English, you need to replace the name in `main.js` and `splatoon3weapon.js`.  

This is a Discord bot written in Node.js that randomly selects weapons compatible with Splatoon 3 Version 11.2.0 on Discord.

## Background of Development

Splatoon 3 doesn't have a feature that lets you choose weapons at random.  
Sometimes I just want to play without being limited to the weapons I usually use.  
That is reason why I made it.

## Features

- `wrc` Command  
This provides a function which select weapon ramdomly.  
There is a mode which you can choose from a limited selection of weapon types.  

## Usage

You can use commands in below:
|Command|Description|Example|
|---|---|---|
|/wrs|Select a weapon ramdomly.|/wrs|
|/wrs &lt;mode&gt;|Select a weapon ramdomly from a limited selection of weapon types.|/wrs mode:shooter_mode|

## Requirements, Dependencies

1. Discord Server
2. Node.js V18 or newer  
3. Discord Bot Token  
You can obtain token from Discord Developer Portal.

## Installation

You'll need a Node.js environment.  
Once you've set that up, use the files in the repository and download the necessary packages according to the package instructions.  

## Setting

To execute, you need to write your Discord token information in the .env file.

## 概要

Discord上でSplatoon 3バージョン11.2.0に存在するブキをランダムに選択してその内容を送信するためのNode.jsで動作するDiscord Botです。

## 開発背景

『スプラトゥーン3』には、武器をランダムに選べる機能がありません。  
たまには、普段使っている武器に縛られずにプレイしたい時もあるでしょう。  
それを解消するためにこれを作ることにしました。

## 機能

- `wrc` コマンド  
これは、ブキをランダムに選択する機能を提供します。  
限られた種類のブキの中から選択できるモードもあります。 

## 使い方

以下のコマンドを使用できます：  
|コマンド|説明|例|
|---|---|---|
|/wrs|ブキをランダムに選択します。|/wrs|
|/wrs &lt;mode&gt;|指定されたブキタイプの中から、ランダムにブキを選択します。|/wrs mode:shooter_mode|

## 前提・依存

1. Discord サーバー
2. Node.js V18 以降  
3. Discord ボットトークン  
トークンは Discord Developer Portal から取得できます。

## 導入方法

Node.js 環境が必要です。  
環境設定が完了したら、リポジトリ内のファイルを使用し、パッケージの指示に従って必要なパッケージをダウンロードしてください。 

## 設定

実行するには、.env ファイルに Discord のトークン情報を記述する必要があります。
